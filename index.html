Colors

This template comes with 8 different color choices. However, if none of those float your boat; changing the highlight color to any other color is pretty simple.

1. go to the "theme-style.css" file and find the hex number of the highlight color. In this example the color is #89ff14 (neon green). We will change it to #FEF900 (yellow)

2. This color is mentioned 23 times in the file so doing a search-and-replace change would be the fastes way. Here is an example using Visual Studio Code:

Text on Navbar

(Optional) change the color of the text sitting over the highlight on the navbar:

  • Go to the tag <nav> </nav> on the top of your pages

  • Change the class text-black to text-white. This step is optional since the black text may read well on the color you chose, however if the color you chose is of a dark hue white text may offer a better contrast. The class you want to change would be next to text-upercase. This is what it looks like:

Google Fonts

Fonts are fun and fortunately they are easy to implement. I am using Goolge fonts, check what they got here. here.

1. Copy the font family or font families' links into the <head> of your html pages. For example, the fonts I am using in this page are Mono Space and Montserrat. The links look like this: 

<link href="https://fonts.googleapis.com/css2?family=Space+Mono&display=swap" rel="stylesheet">
<link href="https://fonts.googleapis.com/css2?family=Montserrat&display=swap" rel="stylesheet">

2. Go to the typography.css file and change the font attributes on the declaration: body, p, li, a, h1, h2, and h3 with the font you linked, They will look something like this: 

  h1 { 
    font-family: "Space Mono", monospace; 
    }
